# 🎬 YouTube Video & Audio Downloader

A **Streamlit-based web app** to download YouTube videos and audio in different formats and resolutions with an easy-to-use interface.

***

## 🚀 Features

*   📥 Download videos in **MP4 format**
*   🎵 Extract audio as **MP3**
*   🎥 Select video resolution (Best, 1080p, 720p, 480p)
*   🖼️ Video preview (title & thumbnail)
*   📝 Custom file naming
*   🔁 Smart retry & fallback system
*   🛠️ 403-error handling improvements

***

## 🧰 Tech Stack

*   **Python**
*   **Streamlit**
*   **yt-dlp**
*   **FFmpeg**

***

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Skillleading-Mohit/Youtube-Downloader-Python.git
cd Skillleading-Mohit/Youtube-Downloader-Python
```

Install dependencies:

```bash
pip install -r requirements.txt
```

***

## ▶️ Run Locally

```bash
streamlit run app.py
```

Open in browser:

    http://localhost:8501

***
